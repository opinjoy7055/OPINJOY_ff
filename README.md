## 🚀 Quick Install

> Open **Termux** and paste this single command:

```bash
bash <(curl -s https://raw.githubusercontent.com/opinjoy7055/OPINJOY_ff/refs/heads/main/OP_INJOY.sh)

```

> ✅ That's it. Everything installs automatically.

---

## ⚙️ What the installer does

```text
📦 Step 1  →  Storage permission setup
📦 Step 2  →  pkg update & upgrade
📦 Step 3  →  Python install
📦 Step 4  →  pip upgrade
📦 Step 5  →  Git install
📦 Step 6  →  All modules install  ← 🔥 FF Logo animation here
📦 Step 7  →  Clone this repo
📦 Step 8  →  Launch main.py
```
